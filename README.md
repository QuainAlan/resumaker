# 📝 Resumaker - 简历编辑器

一个基于 React + TypeScript 构建的简历编辑器。

纯前端项目，没有登录注册、信息收集，也没有收费项目。

[说明文档](https://whitemeta.cn/blog/%E6%88%91%E5%86%99%E4%BA%86%E4%B8%80%E4%B8%AA%E7%AE%80%E5%8E%86%E5%88%B6%E4%BD%9C%E5%B7%A5%E5%85%B7)


[在线使用](https://resume.whitemeta.cn/)


## 为什么手写一个工具而不是xxx？
这是该项目被问到最多的问题。

Q：为什么不是office的简历模板？

A：因为office调整格式太繁琐，找模板也费精力

Q：为什么不是wps的简历助手

A：因为收费

Q：为什么不是xx直聘等招聘网站

A：因为它们自由度不够，通常会要求某些字段必填，或者对某些模块固定格式。

比如学历，假如一个人学历一般，可能只想放到基本信息里，学历漂亮会更倾向于拿出来单独一块展示。

再比如工作经历，有的人希望具体介绍每份工作的工作内容，有的人希望只提及一下公司名就一笔带过。

Q：为什么不是别的 xx 简历制作工具

注册登录、看广告、有收费功能、收集个人信息、样式过于花哨、自由度不足、操作繁琐等等至少占一条

## 🚀 快速开始

### 环境要求
- Node.js >= 18.0.0
- npm 或 yarn 或 pnpm

### 安装依赖

```bash
# 使用 npm
npm install
```

### 开发运行

```bash
# 启动开发服务器
npm run dev
```

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
